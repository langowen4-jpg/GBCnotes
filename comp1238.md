- Babylonian numerals used 2 types of wedges

## Week 2 - Bits and Bytes
- Binary to Decimal: Sum powers of 2 for each '1' bit.
- Hexadecimal to Decimal: Sum powers of 16 for each digit.

## Week 3 - Text Structure
- Markdown is symple syntax for formatting
- HTML is more verbose than Markdown

## Week 4 - Manipulating Text & Text Editors
- * is wildcard for any character sequence
- Regex is for matching, searching, and manipulating text patterns within strings

## Week 5 - Command Line Interfaces
- ls: List files in a directory.
- cd: Change directory.
- pwd: Print working directory.
- cat: Display file contents.
- less: View file contents interactively.
- cp: Copy files.
- mv: Move files.
- mkdir: Create a directory.
- ping: Test network connectivity.
